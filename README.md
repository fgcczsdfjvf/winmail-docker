# Winmail Docker

Winmail 邮件服务器的 Docker 部署配置。

## 部署到 Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

## 环境要求

- Docker
- 80 端口可用
- 至少 512MB RAM
- 至少 1GB 存储空间

## 配置说明

服务会在容器启动时自动安装并运行 Winmail。
可以通过 http://your-domain/ 访问 Winmail 界面。
